CSS3: Estilos internos y externos, selectores CSS

Objetivo General

Diseñar un OVI acerca de CSS3: Estilos internos y externos, selectores CSS.

Objetivos especificos

  • Comprender el manejo de CSS3 para el diseño grafico de las paginas.
  • Explicar la diferencia entre estilos internos y externos y como manipular su uso.
  • Explicar el uso de selectores para el diseño de las paginas web con CSS.

    • CSS (Hojas de estilo en cascada), es una tecnología que nos permite agregar estilo y diseño a un contenido ya realizado en HTML, desde el CSS, podremos cambiar colores, fuentes y tamaños de letras, ubicación de menús, imágenes, artículos, etc.
    (MDN web docs, 2020)

    En este recurso, nos enfocaremos en los conceptos básicos para poder aplicar un diseño con CSS a una página HTML, como lo son los selectores, y el uso de estilos internos y externos.
